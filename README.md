## Getting Started

Building a Simple Chat App With MongoDB & Socket.io

### Installing

1. Must have mongod running: https://docs.mongodb.com/manual/reference/program/mongod/

1. Clone https://github.com/kavinaravind/mongochat.git
2. Npm install
3. Npm start
4. Click on index.html
